Download the exe version of AutoMeet at this link: https://www.dropbox.com/s/yk4rn0rhfov5xl6/AutoMeet.exe?dl=0
To use AutoMeet, put both AutoMeet.exe and MeetInfo.txt in the same file, then edit the parimeters in MeetInfo.txt to join google meet links at a given time.
Finally, run AutoMeet.exe and leave it running until the specified time.
To adjust the location of the mouse click, simply edit the "x" and "y" parimeters in MeetInfo.txt
